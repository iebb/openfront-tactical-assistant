# OpenFront 战术助手 (OF Tactical Assistant)

OpenFront.io 的 Tampermonkey/Greasemonkey 用户脚本，提供全自动战术辅助与中文界面。

## 功能

| 模块 | 说明 |
|------|------|
| **自动出生** | 根据热力图评分选择最优出生点 |
| **自动扩张** | 自动向荒野扩张兵力 |
| **自动农场** | 攻打相邻弱小 AI/国家目标 |
| **自动进攻** | 智能挑选弱势人类玩家发起进攻（仅在 2.5× 优势时出手） |
| **自动防御** | 检测来袭威胁，自动建设防御工事 |
| **自动经济** | 按优先级自动建造城市/港口/工厂 |
| **自动登船** | 自动跨海运兵 |
| **自动武器** | 自动建造并发射核弹/战舰 |
| **自动结盟** | 智能管理外交联盟 |
| **队伍支援** | 自动支援队友 |
| **智能进攻比例** | 根据经济状态限制每次发兵数量，防止过度消耗 |
| **出生热力图** | 可视化地图上的最优出生区域 |
| **顾问面板** | 实时显示威胁/经济/建议 |

## 自动进攻选目标逻辑

基于 [OpenFrontIO 游戏源码](https://github.com/openfrontio/OpenFrontIO) 中的战斗公式：

- **强度比例 ≤ 0.40**：你的有效兵力 ≥ 2.5× 目标，处于最高征服效率
- **经济状态为 safePush**：经济上允许出兵
- **预计占领回合 ≤ 3**：避免拖入消耗战
- 优先选择强度比例最低的目标（最容易击败的）
- 每个目标有独立冷却，防止反复打同一个人造成反弹

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/)
2. 在 Tampermonkey 中新建脚本，将 `ai.user.js` 内容粘贴进去
3. 打开 [openfront.io](https://openfront.io) 即可自动激活

## 界面

- **底部 HUD**：自动化模块开关 + 实时状态
- **左上角 HUD**：日志控制 + 全局开关
- **右侧顾问面板**：出生点排名、威胁列表、建造建议（可拖拽）
- 所有 HUD 的按钮使用 `mousedown: preventDefault`，**不干扰键盘快捷键 1-0**（建造热键）
